PHP之上传文件

2017-11-11

与之前的表单中的$_GET和$_POST一样，在表单中的<input type="file" name="photo">也是个数组，当你确定上传了一张照片或者任何格式的文件后，$_FILES这个数组的内容如下图:

这个数组内有以下属性：

  • $_FILES["photo"]["name"]
    上传的原来的文件名，不代表一个文件，仅仅是一个字符串。
  • $_FILES["photo"]["type"]
    上传的文件类型
  • $_FILES["photo"]["tmp_name"]
    文件上传到服务器的临时的文件名，上传后的文件位于该位置
  • $_FILES["photo"]["error"]
    上传的文件错误信息
  • $_FILES["photo"]["size"]
    上传的文件大小

还有一个相关的函数，就是copy()和move_uploaded_file()，都是把文件从一个地方拷贝到另一个地方，拷贝成功返回true,拷贝失败返回false。两者不同的是，move_uploaded_file()只会拷贝上传的文件，更加安全准确一点相比copy()而言。
